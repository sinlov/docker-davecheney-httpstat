## useage

```bash
$ sudo curl -L --fail https://raw.githubusercontent.com/sinlov/docker-davecheney-httpstat/main/run.sh -o /usr/local/bin/httpstat
$ sudo chmod +x /usr/local/bin/httpstat
# then check
$ httpstat --help
# then check url like
$ httpstat https://www.google.com
```
